# pocket_kirana_store



**pocket_kirana_store** is a Flutter application designed to streamline local grocery shopping. It provides a seamless mobile experience for browsing items from nearby kirana stores, managing a shopping cart, and placing orders with user authentication support.

This project demonstrates Flutter best practices including modular architecture, state management, and integration with backend services for order management.


## Project Structure

```
lib/
├── main.dart
├── screens/
├── widgets/
├── models/
└── utils/
```

## Features

- Browse local grocery items
- Add items to cart
- Manage orders
- User authentication

## Prerequisites

- Flutter SDK 3.0+
- Dart 3.0+
- Android Studio or Xcode

## Installation

1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`

## Dependencies

See `pubspec.yaml` for all dependencies.

## Build

- Android: `flutter build apk`
- iOS: `flutter build ios`
- Web: `flutter build web`
